## Post

Cursor 3 dropped this week.

Agent-first. You write specs. AI writes code. You review outputs.

I use Claude Code for heavy architecture work. Cursor for fast UI iteration. The combo has cut build time down to hours for things that used to take days.

But here's what I keep coming back to:

The hard part isn't prompting. Anyone can prompt.

It's knowing when the AI gave you code that compiles fine but does something wrong.

That requires actually understanding what your product does. What it touches. What breaks if that function is off.

Speed is the easy part. Judgment is the moat.
