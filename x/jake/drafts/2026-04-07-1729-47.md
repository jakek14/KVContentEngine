## Post

I started with Cursor.

Then I added Claude Code for the complex stuff — refactors, multi-file tasks, anything that needed reasoning across the whole codebase.

Now I barely distinguish between them. They run in parallel.

What changed isn't the tools. It's the expectation.

A year ago, AI coding meant autocomplete. Today it means: plan the feature, write the tests, implement it, flag the edge cases, clean up after itself.

The term everyone's starting to use is "agentic engineering." And it's accurate.

The gap between a developer who's internalized this workflow and one who hasn't is already massive.

It's only getting wider.
