## Post

63% of the people building with Claude Code aren't developers.

I'm one of them.

I started in marketing.

Spreadsheets, Canva ads, manual list work.

Now I build and ship web tools for work.

Not because I learned to code — because I learned to spec clearly.

The hardest part of building with AI isn't the AI.

It's knowing exactly what you want to build. What it should do. What it shouldn't. What done looks like.

The tool executes.

The clarity has to come from you.

That's the skill nobody's talking about.
