## Post

1.5 million API keys leaked from a vibe-coded app last month.

No security review. No audit. Just: prompt → deploy → breach.

The tools are real. I use Cursor, Windsurf, Claude Code — 92% of developers do now. 41% of all code being written globally is AI-generated.

The speed is not a gimmick. You can ship 10x faster. That part is true.

But shipping fast without understanding what you shipped is a completely different kind of problem.

You don't need to memorize every function. You do need to know what your app does with credentials. What data it stores. What APIs it exposes to the world.

The vibe coding hangover is hitting the founders who treated "ship it" as the finish line.

Build fast. Just don't deploy a security incident.
